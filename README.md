# Computer Networks Projects

This repository contains projects completed for my Computer Networks coursework. The projects focus on implementing core networking concepts in C/C++, including socket programming, client-server communication, HTTP protocols, and concurrent network services.

## Topics Covered

- TCP/IP socket programming
- Client-server architecture
- HTTP protocol implementation
- Concurrent request handling
- Network communication
- File transfer
- Low-level systems programming

## Projects

### HTTP Server
Implemented a lightweight HTTP server capable of serving static content using TCP sockets while following HTTP protocol specifications.

**Concepts**
- TCP sockets
- HTTP request parsing
- Response generation
- File I/O

---

### Client-Server Communication
Built client and server applications demonstrating reliable communication over TCP sockets.

**Concepts**
- Socket APIs
- Message serialization
- Error handling
- Connection management
